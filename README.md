# BlogApp

**BlogApp** is a simple ASP.NET Core MVC web application for managing blog posts. It allows users to create, view, edit, and delete posts. This project was built to practice ASP.NET Core MVC while applying existing web development skills.

---

## Features

- Create, read, update, and delete blog posts  
- Responsive layout using **Bootstrap 5**  
- Modern typography with **Google Fonts**  

---

## Technologies

- ASP.NET Core MVC  
- C#  
- Entity Framework Core (Code First)  
- Bootstrap 5
